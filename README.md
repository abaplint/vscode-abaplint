# vscode-abaplint
vscode [abaplint](https://abaplint.org) extension

Features:
* Diagnostics / Linting
* Outline
* Hover information
* Go to definition

Keybindings:
* Shift+F1 = pretty print
* Ctrl+F1 = do nothing
* Ctrl+F2 = do nothing
* Ctrl+F3 = save
* Ctrl+Shift+a = open file
* F1 = language help
